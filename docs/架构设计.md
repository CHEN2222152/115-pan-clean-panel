# 115 网盘定时清理面板 — 架构设计

> 编号: project-013
> 创建日期: 2026-07-01
> 状态: 开发中

## 目标

构建一个带 Web 面板的定时任务系统，用于清理 115 网盘中的"最近接收"文件和"回收站"。

## 架构

用户浏览器 → Web 面板 (HTML+CSS+JS) → Flask 后端 API (Python)
    → APScheduler (定时调度)
    → 115 API 客户端
    → 115 网盘 Web API (webapi.115.com)

## 功能

1. **115 Cookie 管理** — 输入/更新 115 登录 Cookie
2. **手动清理** — 立即清理最近接收 / 清空回收站
3. **定时任务** — 配置 cron 表达式定时执行
4. **执行日志** — 查看历史清理记录
5. **OpenList** — 部署 115 文件浏览器

## 技术栈

- 后端: Python Flask + APScheduler + requests
- 前端: 原生 HTML + CSS + JS
- 部署: Docker + docker-compose
