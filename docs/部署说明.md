# 部署说明

## 前提

- 极空间 NAS (192.168.3.254) 已安装 Docker
- SSH 端口 10000，用户 13380599777

## 部署步骤

### 1. 上传文件到极空间

从本地项目目录打包并上传：

`ash
# 在本地打包（排除不需要的文件）
cd D:\codex1\project-013-115网盘定时清理面板
tar czf clean-panel.tar.gz src/ docker-compose.yml

# 上传到极空间
plink.exe -ssh -no-antispoof -batch -hostkey "ssh-ed25519 255 SHA256:mbh50BBVxl/RlPKBDkasN4TAHLiHUBcICfotsw4lhPE" -pw Ctg971217 -P 10000 13380599777@192.168.3.254 "mkdir -p /tmp/clean-panel"
pscp -scp -P 10000 -pw Ctg971217 src/docker/docker-compose.yml 13380599777@192.168.3.254:/tmp/clean-panel/
`

### 2. 构建并启动

`ash
plink.exe -ssh -no-antispoof -batch -hostkey "ssh-ed25519 255 SHA256:mbh50BBVxl/RlPKBDkasN4TAHLiHUBcICfotsw4lhPE" -pw Ctg971217 -P 10000 13380599777@192.168.3.254 "cd /tmp/clean-panel && echo Ctg971217 | sudo -S docker-compose up -d"
`

### 3. 访问面板

- 清理面板: http://192.168.3.254:9532
- AList (文件浏览器): http://192.168.3.254:9533

### 4. AList 配置 115

1. 首次访问 AList: http://192.168.3.254:9533
2. 默认账号/密码: admin / admin
3. 进入管理后台 → 存储 → 添加驱动 → 选择 "115 Cloud"
4. 填写 115 的 Cookie（与清理面板相同）

### 5. 配置清理面板

1. 打开 http://192.168.3.254:9532
2. 从浏览器复制 115 Cookie 粘贴到面板
3. 点击"保存 Cookie" → "验证 Cookie"
4. 配置定时任务 Cron（默认每天凌晨3点）
5. 点击"启用定时"
